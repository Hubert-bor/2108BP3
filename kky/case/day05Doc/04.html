3.包裹节点
	wrap 方法使用指定的 HTML 元素来包裹每个被选元素。
$(selector).wrap(wrappingElement,function(index))
	wrappingElement	必需。规定包裹每个被选元素的 HTML 元素。
				可能的值：
				HTML 元素
				jQuery 对象
				DOM 元素
	function(index)	可选。规定返回包裹元素的函数。
				index - 返回集合中元素的 index 位置。

	unwrap() 	unwrap() 方法移除被选元素的父元素。
			$(selector).unwrap()
	wrapAll() 	wrapAll() 方法使用指定的 HTML 元素来包裹所有被选元素。
			$(selector).wrapAll(wrappingElement)
			wrappingElement	必需。规定包裹被选元素的 HTML 元素。
				可能的值：
				HTML 元素
				jQuery 对象
				DOM 元素
	wrapInner(html) 方法使用指定的 HTML 元素来包裹每个被选元素中的所有内容（innerHTML）。
		$(selector).wrapInner(wrappingElement,function(index))
		wrappingElement	必需。规定包围在每个被选元素的内容周围的 HTML 元素。
			可能的值：
			HTML 元素
			jQuery 对象
			DOM 元素
		function(index)	可选。规定返回包裹元素的函数。
			index - 返回集合中元素的 index 位置。
